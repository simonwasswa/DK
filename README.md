# DK Car Modification - Premium Car Modification Website

A modern, fully responsive car modification company website built with React, TypeScript, Tailwind CSS, Framer Motion, and Supabase.

## ⚡ Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:5173` to see the website live!
